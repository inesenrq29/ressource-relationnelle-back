Dépendances:

Spring Web: pour la création d’API REST

Spring Data JPA: pour l’accès à la base de données

MariaDB Driver: BDD

Spring Security: Authent, gestion des rôles

Validation: validation des formulaires (utile pour la création de compte par exemple)

Lombok: pas de getter/setter et constructeurs

Spring boot DevTools: reload automatique, confort pour le dev
